# 玉环·巧解 — Mechanic Rebuild v2

这是一次从交互底层重新编写的版本，不沿用旧版的“左右滑动 / 经过角度自动解锁”逻辑。

## 本版核心机制

- 玉环圆心固定。
- 手指必须沿玉环圆周转动，使用真实极角拖拽。
- 所有金扣开局全部显示。
- 松手时，玉环缺口必须精确停在金扣中心，才解除一枚金扣。
- 每次手势最多解除一枚金扣，快速乱转不会连续通关。
- 被上层玉环压住的金扣不会提前解除。
- 玉环的全部金扣解除后，玉环自动消散。
- 最后一只玉环消失后正常结算。

## 内容

- 8 个国风珠宝章节。
- 每章 11 关，共 88 个关卡位。
- 每章首关免费试玩。
- 独立系列入口：敦煌、故宫、山海经、二十四节气。
