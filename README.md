# 玉环 · Engine 1.0 核心玩法原型

本版重写了核心操作：

- 拖动玉环管身：旋转开口方向。
- 先点选玉环，再拖动玉环内部空心区域：移动整只玉环。
- 当前最外层玉环的缺口方向正确后，才可以沿金色箭头方向穿扣抽离。
- 金扣固定在结构上，不再自动掉落。
- 采用 900×1100 固定设计坐标，手机端按宽度等比缩放，玉环不会因浏览器高度变化跑出画布。
- 八章第一关均默认免费试玩。
- 每章其余十关按本章进度逐步解锁。
