# 玉环 2.1 · 八玉华章

GitHub Pages 直传版。把本目录中的 5 个文件直接上传到仓库根目录即可：

- index.html
- style.css
- game.js
- bgm.mp3
- README.md

本版重点：

- 修复第二关及所有关卡的解锁判定：目标角度直接按金扣真实位置计算。
- 88 个关卡位，八种玉色，每章 11 关。
- 独立「藏玉阁」选关页面；未解锁关卡显示锁，已通关关卡可重复游玩。
- 金扣真正绘制在两环扣合位置，带金镶玉包边、如意云纹与中央宝钉。
- 玉环加粗，增加半透明、内外缘透光、玉絮和端面。
- 转环清脆碰玉声、解扣声及古风背景音乐。
- 进度保存在浏览器 localStorage。
