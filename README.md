# 玉环 · 巧解重构原型

本版根据用户提供的广告录屏逐帧重新分析后制作，核心变化：

- 不再“角度对准后金扣自动掉落”。
- 玉环可直接移动与旋转。
- 缺口必须靠近金扣，并沿离开金扣的方向拖动，才会真正穿扣。
- 金扣固定留在原结构中。
- 玉环完全脱离所有金扣并拖出棋盘后才算取下。
- 八章第一关均可免费试玩。
- 手机使用固定 900×1120 设计坐标等比缩放。

## GitHub 上传

把以下文件上传到仓库根目录并覆盖同名文件：

- index.html
