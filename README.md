# 玉环 3.1 · 巧解重构版

这是全新的二维空间拆解原型，不再使用“缺口对准后金扣自动掉落”的旧玩法。

## 核心操作

1. 点击玉环。
2. 点击“转环”，沿玉环圆周拖动，让 C 形开口朝向棋盘外侧。
3. 点击“拖环”，将已经显露通道的玉环向外缓缓抽离。
4. 只有当前结构中真正可以拆出的玉环能够离开；顺序错误或通道未对准时会回弹。
5. 逐层抽离，直至全部解开。

本版为浏览器可运行的 2D 空间拆解模拟。它已经完成“旋转 + 拖拽 + 顺序约束 + 抽离”的核心循环，但并非三维刚体物理引擎。后续可在这套交互上继续增强环与环、环与扣之间的精确碰撞路径。
