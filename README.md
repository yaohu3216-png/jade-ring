# 玉环·巧解（Editable Prototype v0.3）

这是一个无需安装依赖、可以直接修改的 HTML5 Canvas 原型。

## 运行

直接双击 `index.html`，或上传整个文件夹到 GitHub Pages / Netlify / Cloudflare Pages。

## 你最常修改的地方

- `levels.js`：章节名称、颜色、关卡、玉环坐标、金扣关系。
- `style.css`：首页、按钮、章节卡片、UI 外观。
- `game.js`：旋转、解扣、消失动画、Canvas 材质。
- `assets/art-direction.png`：本次生成的美术方向图，仅作为首页背景与设计参考。

## 关卡数据

### 玉环

```js
{id:'A', x:450, y:220, r:112, gap:-1.45, z:5}
```

- `x/y`：中心坐标
- `r`：半径
- `gap`：初始缺口角度（弧度）
- `z`：前后层级

### 金扣

```js
{id:'AB', a:'A', b:'B', unlock:'A', after:['X'], diamond:true}
```

- `a/b`：连接的两只玉环
- `unlock`：缺口需要通过金扣的玉环
- `after`：这些环消失后，该扣才可以解开；即使未激活也会从开局显示
- `diamond`：是否有小钻
- `offset`：调整金扣沿连接方向的角度

## 当前机制

- 所有金扣从开局完整显示。
- 玩家只旋转玉环，不拖出棋盘。
- 缺口必须完整越过金扣中心才解除。
- 一次手势最多解除一扣。
- 玉环解除全部有效金扣后自动消失。

此版本为可编辑核心原型，不代表最终商业美术或完整 88 关。
